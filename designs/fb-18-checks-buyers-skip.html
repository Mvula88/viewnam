Most Buyers Skip These

5 checks to do
before you pay.

Save this post. Even if you never book us — these checks alone will protect you.
🔎
Cold start the engine yourself
If it's been pre-warmed, ask why. Hidden issues usually show on cold start.
📄
See the NaTIS printout
Not a copy. The original. Check for Code 2, 3, or 4.
🔢
Match 3 VINs
Dashboard, engine bay, door sticker — all three must match the papers.
🛢️
Pull the oil dipstick
Milky = head gasket. Black paste = neglect. Clean amber = cared for.
🛣️
Test drive alone (if possible)
A chatty seller is distracting you from strange noises.
