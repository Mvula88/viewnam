Before You Pay
Save this post.
It will save you money.

Five checks
most buyers skip.

i.

Cold start the engine yourself

If it's pre-warmed, ask why. Problems show on cold start.

ii.

See the NaTIS printout

Not a copy. The original. Check for Code 2, 3, or 4.

iii.

Match three VINs

Dashboard. Engine bay. Door sticker. All must match the papers.

iv.

Pull the oil dipstick

Milky means head gasket. Black paste means neglect.

v.

Test drive alone, if possible

A chatty seller is distracting you from strange noises.
